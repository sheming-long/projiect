# git是什么
分布式版本控制系统(dvcs)=>svn
集中式版本控制系统(vcs)=>git



# git 的作用
在项目的开发过程中,对值得记录的时间节点进行备份,方便后期恢复   (后悔药)

方便团队协作开发



# git 的三种状态
git 有三种状态 ,你的文件可能处于其中之一
-   已修改   (modeified)
-   已暂存   (staged) 
-   已提交  (committed):表示数据已经保存在本地仓库
